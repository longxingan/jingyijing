# 静一静 (JingYiJing) v9.0.9

> Windows 专注助手 —— 全屏锁屏冥想、喝水提醒、休息提醒，助你保持专注与健康。

---

## 项目简介

**静一静** 是一款专为 Windows 平台设计的桌面专注助手。它通过全屏锁屏、定时提醒、数据统计等方式，帮助用户在工作中保持专注，并养成良好的喝水和休息习惯。

适用于需要长时间面对电脑的上班族、学生、自由职业者等人群。

---

## 功能特性

- [x] 全屏锁屏冥想（倒计时解锁，ESC 多次可强制退出）
- [x] 喝水提醒钟（可设置间隔、锁屏时长、自定义提醒文字）
- [x] 休息提醒钟（可设置间隔、锁屏时长、自定义提醒文字）
- [x] 工作时间设置（支持午休时段、休息日配置）
- [x] 全局热键锁屏（Alt+P）和显示/隐藏窗口（Alt+O）
- [x] 系统托盘图标（窗口隐藏到托盘，双击/右键菜单操作）
- [x] 开机自启（注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`）
- [x] 看门狗进程守护（主进程崩溃后自动重启）
- [x] 单实例检测（互斥锁防止多开）
- [x] 自定义语录系统（JSON 存储，支持去重、随机展示）
- [x] 数据统计（今日喝水/休息次数、连续打卡天数、本周峰值、图表可视化）
- [x] 数据导出 CSV
- [x] 多主题切换（ttkbootstrap 13 套主题）
- [x] 锁屏界面字体大小可调

## 安装方法

### 环境要求

- Windows 11（推荐）
- Python 3.12

### 从源码运行

```bash
# 克隆仓库
git clone https://github.com/longxingan/jingyijing.git
cd jingyijing

# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

### 直接运行 exe

下载 `静一静_v9.exe`，双击运行即可，无需安装 Python 环境。

---

## 使用方法

1. **启动程序**：运行 `python main.py` 或双击 `静一静_v9.exe`
2. **立即锁屏**：点击首页的「立即锁屏」按钮，或按全局热键 `Alt+P`
3. **设置提醒**：切换到「提醒」页面，配置喝水钟和休息钟的间隔时间
4. **管理语录**：切换到「语录」页面，添加你喜欢的专注语录
5. **查看统计**：切换到「统计」页面，查看今日数据和趋势图表
6. **个性化设置**：切换到「设置」页面，选择主题、调整锁屏参数、开启开机自启

---

## 快捷键说明

| 快捷键 | 功能 |
|--------|------|
| `Alt + P` | 全局锁屏（立即进入全屏倒计时） |
| `Alt + O` | 显示/隐藏主窗口（最小化到托盘/恢复） |
| `ESC` x3 | 锁屏状态下强制退出（默认按 3 次） |

> 热键可在「设置」页面自定义。

## 打包方法

使用 PyInstaller 打包为单文件 exe：

```bash
# 安装 PyInstaller
pip install pyinstaller

# 单文件模式打包
pyinstaller --name 静一静_v9 --onefile --windowed --noconfirm --clean \
  --collect-data ttkbootstrap --collect-data pystray \
  --add-data "quotes.json;." main.py
```

或使用项目自带的打包脚本：

```bash
python build.py
```
